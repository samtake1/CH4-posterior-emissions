# CH4-posterior-emissions

Gridded (0.1° × 0.1° latitude–longitude, monthly) and national-scale monthly total posterior CH₄ emission datasets for South Korea, covering the period 2010–2021, together with the corresponding prior emissions, derived from the STILT inversion are provided in this repository. The inversion utilizes in-situ observations from two WMO/GAW sites in South Korea: Anmyeondo and Gosan. For any inquiries regarding the data or related information, please feel free to contact us. We welcome collaboration and potential co-authorship for studies that make substantial use of these data products. Updated versions of the emission datasets will continue to be released through this repository.

Data reference:

Kenea, S. T., Shin, D., Lee, S., Wang, F., Maksyutov, S., Joo, S., Lee, S., Kim, S. Methane (CH₄) Emission Estimates for South Korea using Lagrangian-Based Inversion Modeling (2010–2021). http://dx.doi.org/10.2139/ssrn.5812026, in review, 2026.
